qml-curatorim

![stream](http://i.imgur.com/PoIvYMq.png)

## Running on Ubuntu 

    $ sudo apt-get install qmlscene qtdeclarative5-xmllistmodel-plugin \ 
    qtdeclarative5-window-plugin qtdeclarative5-qtquick2-plugin
    $ qmlscene ./main.qml
